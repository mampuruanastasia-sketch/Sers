# **App Name**: SERS: Student Emergency Response Solution

## Core Features:

- Incident Reporting: Allow students to report incidents (Fire, Medical, GBV, Bullying, Crime) with detailed descriptions, location, and media uploads.
- Profile Creation: Enable students to create profiles with necessary contact and emergency information, securely stored in Firestore.
- Real-time Alerts: Notify campus staff instantly via SMS and in-app notifications upon receiving a new incident report.
- Admin Dashboard: Provide a dashboard for campus staff to view, manage, and respond to incident reports.
- Location Services: Integrate location services to automatically detect and display the student's current location when reporting an incident.

## Style Guidelines:

- Primary color: Deep crimson (#ED3F27), for conveying the app's mission of urgent responsiveness; dark scheme.
- Accent color: Vivid orange (#FEB21A), for focus and highlight.
- Secondary color: Light beige (#FDF4E3) for backgrounds and UI elements requiring less emphasis.
- Neutral color: Dark navy (#134686) for text and icons to ensure readability and contrast.
- Body and headline font: 'Inter', sans-serif.
- Use clear, universally recognizable icons for incident types (fire, medical, etc.).
- Design a clean, intuitive layout with high contrast for ease of use in emergency situations.
- Incorporate subtle animations to guide users through the reporting process.
